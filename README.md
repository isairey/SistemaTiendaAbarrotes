# 🛒 Sistema de Abarrotes — Java

Aplicación de escritorio desarrollada en **Java** para la gestión de una tienda de abarrotes, permitiendo controlar ventas, inventario, productos y usuarios de manera eficiente.

---

## 🚀 Descripción

El **Sistema de Abarrotes** está diseñado para pequeños y medianos negocios que necesitan una solución práctica para administrar sus operaciones diarias.

Permite registrar ventas, controlar el inventario en tiempo real y llevar un seguimiento de productos, optimizando la administración del negocio.

---

## 🎯 Características principales

* 🧾 **Ventas**

  * Registro de ventas rápidas
  * Cálculo automático de totales
  * Generación de tickets

* 📦 **Inventario**

  * Control de existencias
  * Actualización automática al vender
  * Alertas de productos bajos

* 🏷️ **Gestión de productos**

  * Alta, edición y eliminación
  * Clasificación por categorías

* 👤 **Usuarios**

  * Inicio de sesión
  * Roles básicos (Administrador / Cajero)

* 💰 **Caja**

  * Control de ingresos diarios
  * Historial de movimientos

* 📊 **Reportes**

  * Ventas por día
  * Productos más vendidos

---

## 🛠️ Tecnologías utilizadas

* ☕ Java (JDK 8+)
* 🖥️ Swing / JavaFX (interfaz gráfica)
* 🗄️ MySQL / SQLite (base de datos)
